Compatibility list

This list includes all assets than can be tracked in Google Sheets via muFunds.

Many real-time data assets (such as stocks or ETFs) are compatible with Google Finance, and in these cases we recommend the Google Finance function included in Google Sheets, that usually provides reliable data with small delays. The list of exchanges connected to Google Finance is available in their disclaimer page.

For all the other assets (mutual funds, pension plans, bonds, etc.), muFunds relies in different data sources listed below, with the codes that should be used as the source argument for the muFunds function.

Note: Search compatible sources and identifiers for the country where the asset is available for sale, not where the asset is registered. For example, data for mutual funds from Luxembourg is available in most European Morningstar pages.
