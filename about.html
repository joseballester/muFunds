µFunds

Authors: José Ballester and the µFunds community at GitHub.

Version: v2.4

µFunds lets you import your mutual fund NAVs and other attributes from Morningstar and other sources to your Google Sheets spreadsheet by using a simple formula.

Usage

Start using µFunds by typing the following function in any cell:

=muFunds(option, id, source)

Learn about these arguments at the Usage and examples documentation page.

For a list of available countries, assets, sources and identifiers check the Compatibility list.
