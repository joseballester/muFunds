muFunds

Authors: Jose Ballester and the muFunds community at GitHub.

muFunds lets you import your mutual fund NAVs and other attributes from Morningstar and other sources to your Google Sheets spreadsheet by using a simple formula.

Usage

Start using muFunds by typing the muFunds function in any cell. For example, to check the NAV of the Vanguard Global Stock Eur Acc mutual fund (with ISIN IE00B03HD191), use:

=muFunds("nav", "IE00B03HD191")

Learn about the function arguments at the Usage and examples documentation page.

For a list of available countries, assets, sources and identifiers check the Compatibility list.
